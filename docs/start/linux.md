**Цель раздела** — познакомиться c **Linux** и научиться **работать в консоли:**

- ориентироваться в файловой системе,
- подключаться к другим компьютерам в сети,
- знать основные команды bash.

## Кратко о Linux

Linux — это семейство операционных систем, основанных на ядре «Linux», написанном на языке программирования C (Си). Операционная система управляет всеми ресурсами компьютера — процессами, памятью, файлами, устройствами — и обеспечивает интерфейсы взаимодействия между пользователем, программами и «железом». В основе Linux лежит идея открытого кода и модульности: ядро отвечает за базовые функции, а поверх него устанавливаются утилиты, библиотеки и графические оболочки. Вместе всё это формирует дистрибутив — полноценную операционную систему (например, Ubuntu, Debian, Kali Linux).

Для специалиста по информационной безопасности Linux — основной инструмент. Здесь важна работа не с графикой, а с терминалом и оболочкой (shell), которая позволяет напрямую управлять системой с помощью текстовых команд. Понимание структуры файловой системы, умение находить и редактировать файлы, настраивать права доступа и подключаться к другим устройствам — базовые навыки для любого, кто участвует в CTF или работает с кибербезопасностью.

## Подготовительное задание

### 1. Виртуальная машина

**Виртуальная машина (ВМ)** — это как **отдельный компьютер внутри вашего компьютера**. Она запускается с помощью специальной программы — примерно как *Word* открывает документы или *PowerPoint* показывает презентации, только эта программа «открывает» целый компьютер, на котором можно установить другую операционную систему, например, Kali Linux, Ubuntu, или даже вторую копию Windows.

Специальные программы для запуска виртуальных машин называются ***гипервизорами***. Наиболее известными из них являются VirtualBox и VMware Workstation Player.

В качестве операционной системы (ОС) на ВМ будем использовать **Kali Linux**. Почему именно эту ОС? Потому что Kali Linux — это популярный дистрибутив для поиска и эксплуатации уязвимостей, содержащий широкий спектр инструментов на эту тему. Он отлично подходит для участия в CTF.

!!! abstract "Описание задания"
    
    Познакомьтесь с основами виртуализации и настройте **виртуальную машину** на своём компьютере. Используйте гипервизор (например, VirtualBox) для запуска готового образа **Kali Linux**.

Этапы выполнения:

1. **Изучите тему виртуализации**, виртуальных машин и гипервизоров. Информацию можно найти в Интернете.
2. **Установите на свой компьютер гипервизор**. Рекомендуется использовать **VirtualBox**. Подробные инструкции по установке (текстовые или видео) доступны онлайн.
3. **Скачайте дистрибутив** предустановленной виртуальной машины **Kali Linux** с официального сайта по [ссылке](https://www.kali.org/get-kali/#kali-virtual-machines) и запустите её в гипервизоре. Процедуру импорта и запуска виртуальной машины также можно найти в Интернете.
