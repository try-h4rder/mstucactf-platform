# Веб-технологии
**Цель раздела** — познакомиться с основами **веб-технологий** и принципами работы **протокола HTTP**.

## Web

Современный веб основан на **клиент-серверной архитектуре**, где клиент (обычно браузер) отправляет запросы к серверу, а сервер отвечает, передавая нужные данные. Основой взаимодействия между ними является протокол **HTTP (HyperText Transfer Protocol)** — прикладной протокол передачи гипертекста, то есть веб-страниц, изображений, скриптов и других ресурсов. HTTP определяет структуру запросов и ответов, а также правила обмена данными между браузером и сервером.

Каждый HTTP-запрос содержит метод (например, **GET** для получения данных или **POST** для их отправки), путь к ресурсу, версию протокола и набор заголовков. Ответ сервера состоит из кода состояния (например, 200 — успех, 404 — страница не найдена), заголовков и тела с содержимым страницы. На основе этого обмена формируется работа всего веб-пространства — от простых сайтов до сложных веб-приложений.

Язык **HTML (HyperText Markup Language)** используется для разметки веб-страниц, позволяя браузеру интерпретировать и отображать информацию в виде текста, изображений, ссылок и интерактивных элементов. Веб-технологии объединяют принципы сетевого взаимодействия, структурирования данных и безопасности, формируя фундамент современного Интернета.
